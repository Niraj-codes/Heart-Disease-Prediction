# Heart Disease Prediction using Machine Learning

## Project Overview

This project predicts whether a person has heart disease using machine learning classification algorithms.

## Dataset

Source:
https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

## Algorithms Used

- Logistic Regression
- Random Forest Classifier
- K-Nearest Neighbors (KNN)

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Preprocessing
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Best Model Analysis

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score

## Best Model

Logistic Regression achieved the best performance with an accuracy of **80.33%**.
